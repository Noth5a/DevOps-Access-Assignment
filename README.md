To sign up for an account select sing up in the nav bar, enter an email, name and matching passwords. Then select whether you would like a manager or user account with the check box. This will automatically log you in. if you need to log in again you will need to select log in on the nav bar and then enter your email and password. 

Once logged in if you are a manager you can see all the Requests you have added with their email, access, created on date and who added them. if you are a user you will only see the request for your access and Admins will see everyone's requests. 

To add a new Request add the requested for email and access level. Then select Add Request and click home to refresh the page. 

To edit a Request click Update on the Request you wish to edit, a popup should appear prefilled with the information. Simply edit what you need to and select save changes. To close this popup select close. 

To delete a Request select the cross. if you are not an admin you will receive a popup telling you that you cant delete. 

To logout select logout on the navbar.
