To sign up for an account select sing up in the nav bar, enter an email, name and matching passwords. Then select whether you would like an admin or user account with the check box. This will automatically log you in. if you need to log in again you will need to select log in on the nav bar and then enter your email and password. 

Once logged in you can see all the certifications you have added with their name issuer issued date expiry date and who added them. if you are an Admin you will see everyone's certifications. 

To add a new certification add the name, issuer, issued date and expiry date. Then select Add Certification and click home to refresh the page. 

To edit a certification click Update on the certification you wish to edit, a popup should appear prefilled with the information. Simply edit what you need to and select save changes. To close this popup select close. 

To delete a Certification select the cross. if you are not an admin you will receive a popup telling you that you cant delete. 

To logout select logout on the navbar.
